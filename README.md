# SmartMine Maintenance System

SmartMine Maintenance System is a web-based dashboard designed to monitor mining equipment health and support preventive maintenance.

## Features
- Equipment health dashboard
- Status indicators (Good / Warning / Critical)
- Backend API using Flask
- Dynamic frontend rendering using JavaScript

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- API Communication: REST
- Security Handling: CORS (Flask-CORS)

## What I Learned
- How frontend and backend communicate using REST APIs
- How to build a backend using Flask
- How to fetch and render data dynamically using JavaScript
- How browser security (CORS) affects API communication
- How to debug real-world errors using browser DevTools

## Challenges Faced
- Connecting frontend and backend running on different ports
- Handling CORS errors while fetching API data
- Debugging UI rendering issues caused by JavaScript logic
